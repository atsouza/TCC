1 - Para adicionar um critério ou um subcritério basta clicar no botão "+Critério".

2 - Para adicionar um subcritério do critério supra citado, basta clicar no mesmo botão "+Critério" e preencher o campo seguinte com os dados do subcritério que deseja adicionar. Isso se mantém para todos os subcritérios.

3 - Ao terminar os subcritérios do item, para adicionar outro item, basta clicar em "+Critério" e preencher o próximo campo com os dados do próximo critério. Se este item tiver subcritérios, basta seguir os passos do item anterior.
